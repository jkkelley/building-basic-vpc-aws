### Steps to Follow

#### Create VPC

- [ ] Login to AWS Account

- [ ] Head to VPC and select "Your VPCs" from the left menu bar.
    - [ ] Click on "Create VPC" button
    - [ ] Fill in Name tag (Optional)
    - [ ] Fill in IPv4 CIDR Block
        - [ ] Choose IPv6 CIDR Block (Optional)
    - [ ] Click create button
